# 📋 TaskNest

A modern, responsive task management web application built using HTML, CSS, and Vanilla JavaScript.

Manage your daily tasks with ease using features like search, filters, sorting, dashboard analytics, dark mode, toast notifications, and persistent local storage.

---

## 🌐 Live Demo

The live demo will be available after deployment using GitHub Pages.


---

## 📖 About

TaskNest is a responsive task management application designed to help users organize their daily work efficiently.

The project focuses on applying core front-end development concepts including DOM manipulation, event handling, Local Storage, responsive web design, reusable components, and clean JavaScript architecture without relying on external frameworks.

---

## ✨ Features

- ➕ Add new tasks
- ✅ Mark tasks as completed or pending
- 🗑 Delete tasks
- 🔍 Search tasks instantly
- 🎯 Filter by status
- 🚩 Filter by priority
- ↕ Sort tasks by newest, oldest, alphabetical order, or due date
- 📊 Dashboard with statistics
- 📈 Animated progress bar
- 🌙 Dark mode with saved preferences
- 🔔 Toast notifications
- ✔ Form validation
- 💾 Local Storage support
- 📱 Fully responsive design
- 🎨 Smooth UI animations

## 🚀 Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/TaskNest.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional dependencies or installations are required.

---

## 💻 Usage

- Add a new task with a priority and due date.
- Search tasks instantly using the search bar.
- Filter tasks by status or priority.
- Sort tasks based on different criteria.
- Mark tasks as completed or pending.
- Delete tasks when no longer needed.
- Switch between Light and Dark Mode.
- Refresh the page—your tasks and selected theme are saved automatically.
